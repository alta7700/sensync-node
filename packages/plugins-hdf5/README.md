# packages/plugins-hdf5

Пакет с recorder-плагином для записи выбранных `signal.batch` потоков в `HDF5`.

## Для чего

- Даёт отдельный runtime-плагин записи без смешивания логики хранения с `ui-gateway`, fake-адаптерами или ядром runtime.
- Фиксирует первый рабочий путь записи через `h5wasm`.

## Как работает

- Плагин слушает команды `recording.start / pause / resume / stop`.
- Во время записи он принимает только выбранные `channelId`.
- Для каждого канала плагин копит JS-буферы и сбрасывает их в `HDF5` по порогам.
- На `pause` и `stop` делает принудительный flush.

## Взаимодействие

- Использует контракты из `packages/core`.
- Запускается через `packages/plugin-sdk` как обычный worker-плагин.
- Может подключаться к любым адаптерам/процессорам, которые эмитят `signal.batch`.
