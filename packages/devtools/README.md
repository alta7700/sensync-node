# packages/devtools

Заготовка под bench и инспекторы.

## Для чего

- Пакет зарезервирован под synthetic load, debug-инспекторы и утилиты профилирования.

## Как работает

- В текущем `v1` здесь только заглушка, без реализованных инструментов.

## Взаимодействие

- При развитии должен использовать публичные контракты `packages/core` и runtime API, а не залезать во внутреннее состояние приложений.
